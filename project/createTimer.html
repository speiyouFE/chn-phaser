    # 创建定时器

    创建定时器 - 倒计时（左上角倒计时）

    // 引入函数
import { createTimer } from './tools'

// 调用函数
createTimer(game, x, y, time, Textstyle, inDurationTime, delayTime, sound)

    # 参数

    参数说明类型
    game创建到指定的game，通常是全局的gamestring
    x相对于整个舞台的 x 坐标number
    y相对于整个舞台的 y 坐标number
    time时间[ms]number
    Textstyle样式名称string
    inDurationTime出场动画持续时间number
    delayTime动画执行延迟时间number
    sound倒计时声音string
